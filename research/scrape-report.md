## PAGE STRUCTURE NOTES

**LG Nepal category page structure:**
1. Hero carousel (full-bleed, 1600×600px banners — promotional, text-heavy, DO NOT use as product images)
2. Sub-category navigation strip (horizontal tiles with category names)
3. Product listing grid (3 columns desktop, cards with 350×350px thumbnails, model name, SKU, CTA)
4. Feature banners (split sections, product + copy alternating)
5. "You may also like" cross-sell strip at bottom

**Product Detail Page (PDP) structure:**
1. Gallery (main image + thumbnail strip — typically 12–17 images)
2. Product title + SKU
3. Size/variant selector
4. Key Features accordion
5. "Where to Buy" CTA (primary) + "Add to Compare"
6. Full spec table below fold
7. Feature detail sections (expandable)

**Gallery image naming conventions by category:**
- Refrigerators: `/GALLERY/DZ-1.jpg` (uppercase Gallery, numbered 1–9)
- Washing Machines: `/gallery/DZ-1.jpg` (lowercase gallery, numbered 1–N)
- Air Conditioners: `/Gallery/DZ-01.jpg` (titlecase Gallery, zero-padded)
- TVs: `/gallery/large02.jpg` or `/galleryy/large02.jpg` (note typo on some QNED models — double y)
- Soundbars: `/gallery/Z01_large01.jpg` (prefixed with Z0N_)
